# HumTech.ai Website

Official website for HumTech.ai - Fractional AI Directors for SMEs.

## About

HumTech provides three experienced AI Directors for a fraction of the cost of hiring one full-time AI Director. We deliver enterprise-grade AI leadership without the enterprise price tag.

## Tech Stack

- **Astro** - Static site generator
- **TailwindCSS** - Utility-first CSS framework
- **TypeScript** - Type-safe JavaScript

## Development

### Prerequisites

- Node.js 18+
- npm or pnpm

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

Visit `http://localhost:4321` to view the site.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
